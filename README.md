# Walkie-Talkie App

A Flutter application for offline walkie-talkie communication using nearby connections and USDT payments via QR code.

## Features

- Offline peer-to-peer audio communication
- USDT payment screen with QR code
- Premium dark/gold UI theme

## Setup

1. Install Flutter SDK from https://flutter.dev
2. Run `flutter pub get` to install dependencies
3. For Android, ensure permissions in AndroidManifest.xml
4. Run `flutter run` to launch the app

## Dependencies

- nearby_connections: For offline P2P
- qr_flutter: For QR code generation
- flutter_sound: For audio recording/playback
- permission_handler: For permissions
